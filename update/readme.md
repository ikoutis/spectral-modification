This folder contains the original paper and more recent updates to the work.

Current working artifacts:

- `spectral-maximizer.tex`: rigorous manuscript draft for spectral maximizers,
  Cheeger inequalities, and cut-preserving rewiring algorithms.
- `references.bib`: BibTeX file for the manuscript.
- `algorithm-notes.md`: implementation roadmap for exact implicit maximizers,
  sparse edge sampling, certificates, and experiments.
- `literature/`: cached open-access PDFs and a literature index.
